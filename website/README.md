# Kopi Kenangan Website

Website resmi Kopi Kenangan yang dibangun sebagai landing page modern, responsif, dan production-ready dengan fitur utama:

- Hero section yang menarik
- Halaman menu dengan filter kategori
- Promo dan outlet informasi
- Order online sederhana dengan cart interaktif
- Membership dan contact form
- SEO dasar melalui sitemap dan robots.txt

## Jalankan Lokal

Buka folder website lalu jalankan server statis:

```bash
python3 -m http.server 3000
```

Lalu buka http://127.0.0.1:3000
