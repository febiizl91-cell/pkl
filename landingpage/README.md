# CasualStep

Implementasi static web app berdasarkan `prd5.md`: toko online sepatu casual dengan katalog, filter, detail produk, wishlist, cart, checkout, order history, dan admin dashboard demo.

## Jalankan Lokal

```bash
cd casualstep
python3 -m http.server 3001
```

Buka `http://127.0.0.1:3001`.

## Admin Demo

- Username: `admin`
- Password: `step123`

Data produk, cart, wishlist, dan pesanan disimpan di `localStorage` browser.
