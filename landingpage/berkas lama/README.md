# TroubleMaker

Implementasi static web app toko online jaket streetwear dengan katalog, filter ukuran, detail produk, login/register, wishlist, cart, checkout, riwayat pesanan, dan admin dashboard demo.

## Jalankan Lokal

```bash
cd landingpage
python3 -m http.server 3001
```

Buka `http://127.0.0.1:3001`.

## Admin Demo

- Username: `admin`
- Password: `tm12345`

Data produk, akun customer, cart, wishlist, dan pesanan disimpan di `localStorage` browser.
