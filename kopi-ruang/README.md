# Kopi Ruang

Proyek website Kopi Ruang.

## Struktur

- `app.py` - backend Flask sederhana
- `templates/` - HTML template
- `static/` - CSS, JavaScript, aset
- `data/` - database SQLite

## Jalankan

1. Install Python 3.12
2. Buat virtualenv: `python3 -m venv venv`
3. Aktifkan: `source venv/bin/activate`
4. Install: `pip install -r requirements.txt`
5. Jalankan: `python app.py`

## Tujuan

Menerapkan frontend, backend, autentikasi, dan halaman sesuai PRD.
